Welcome to Baystation

We're a Space Station 13 community and server with a focus on action and character play.

In particular, we value sportsmanship, positive attitudes, and inclusive behavior.

Space Station 13 is an open source multi-player game with rich mechanics built on BYOND.

You can learn more about Space Station 13 through the SS13 reddit community.

Connection Information

Use a 514 client. 513 will crash. 515 compatibility is incomplete.

You can find the latest 514 at:

  • The current stable channel update through the BYOND pager.
  • The stable download at www.byond.com/download.
  • Or the last build in www.byond.com/download/build/514.

Connections from builds older than 1585 are prevented.

We recommend that clients run build 1589.


Other Notes

You may find character slot names not showing in menu lists.
This is cosmetic: re-save the slot to show the name correctly.
